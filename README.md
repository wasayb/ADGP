# ADGP Power Solutions Website

A **premium, scroll-driven, cinematic single-page website** for **ADGP Power Solutions (Al-Wasay Diesel & Gas Power)**.  
Designed to showcase **industrial generators** (brand new and refurbished) and highlight the company’s **global import/export operations** in Pakistan, Dubai, Russia, Holland, and worldwide.

---

## Features

- **Cinematic Scroll Storytelling:** Smooth scroll with layered parallax for a high-end, immersive experience.
- **Industrial Luxury Design:** Dark steel palette with muted gold accents, clean typography, and professional layouts.
- **Contact Integration:** Direct WhatsApp links for CEO and Operations/Sales Head.
- **Global Operations Showcase:** Highlighting markets and export services.
- **Micro-Interactions & Ambient Sound:** Subtle hover effects and optional industrial ambient audio for immersion.
- **AI Studio Ready:** Fully runnable in **AI Studio** using HTML, CSS, and JavaScript — no WebGL required.
- **Responsive & Optimized:** Designed to run smoothly on both desktop and mobile.

## Installation / Usage

1. Open the project in **AI Studio**.  
2. Paste the generated HTML, CSS, and JS files.  
3. Preview and interact directly inside AI Studio.  
4. Optional: Export to web hosting (Vercel, Netlify, etc.) for public access.

---

## Technology Stack

- **HTML / CSS / JavaScript** – Core web technologies  
- **AI Studio (Gemini 3)** – Generates site structure, scroll logic, and micro-interactions  
- **Parallax & Scroll Animations** – CSS & JS based  
- **Responsive Design** – Mobile & desktop compatible

## Project Goal

To create a **high-trust, global B2B website** that communicates authority, reliability, and premium industrial solutions without relying on WebGL or heavy frameworks. Perfect for showcasing ADGP Power Solutions’ **brand, products, and international presence**.
